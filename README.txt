A man vs machine game.
Pacman is back and now the ghosts has to flee.
The game goal is to catch all the ghosts to make them white.
Get what THEY think and winning is on your way, try chasing them withuot a plan, and you might find yourself endlessly running... 
-based on UC Berkeley code for the pacman game- 

run via python 3 with inputs -g CatAgent -l <layout>
example:
>> py3 -g CatAgent -l simplePlus2
